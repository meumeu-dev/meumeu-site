meumeu.dev / hop

hop Beta

Un seul binaire pour gerer toutes tes machines. Sandboxe par defaut — zero trace au reboot. SSH, tunnels, pairing chiffre E2E.

Install

$ curl -sL meumeu.dev/hop/install | bash

Linux & macOS (amd64, arm64, arm32 Linux). Mise a jour: hop update

PS> iwr -useb meumeu.dev/hop/install.ps1 | iex

Windows (amd64) — ouvre PowerShell, pas de double-clic sur le .exe (c'est un CLI).

Features

🚫
Sandbox par defaut

Config dans /tmp, disparait au reboot. hop exit = zero trace. hop install pour rendre permanent.

🔗
Pairing chiffre E2E

LAN direct ou relay Cloudflare. AES-GCM + Argon2id. Le relay ne voit jamais tes donnees.

🔌
SSH intelligent

Auto-detecte LAN ou tunnel Cloudflare. Toujours le chemin le plus rapide.

🌐
Cloudflare Tunnels

Acces distant sans ouvrir de port. Foreground par defaut, service systemd avec hop install.

⚡
Services & Config

Commandes reutilisables, aliases. Export/import chiffre. hop config cf pour tout configurer.

🧹
Tiny & portable

Binaire unique, zero dependances. Self-update + checksum SHA256. Autocompletion bash/zsh/fish.

Quick start

1
Install & init

curl -sL meumeu.dev/hop/install | bash && hop init

2
Pairer une machine

hop pair — sur l'autre: hop pair <token>

3
Se connecter

hop ssh rpi — mode sandbox, zero trace au reboot

4
Garder ou quitter

hop install pour rendre permanent — ou hop exit pour tout effacer
